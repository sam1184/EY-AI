üöÄ Amazon Bedrock Overview

Serverless Foundation Models for Enterprise AI

What is Amazon Bedrock?

üè¶ SecureBank GenAI Transformation

Before Bedrock:

  • SageMaker Complexity: Weeks to deploy LLMs
  • Infrastructure Management: EC2, scaling, monitoring
  • Model Training: Expensive, time-consuming
  • Limited Models: Build from scratch

After Bedrock:

  • Serverless: Deploy in minutes, no infrastructure
  • Fully Managed: AWS handles scaling, updates
  • Pre-trained Models: Claude, Llama, Cohere ready
  • Pay-per-use: $0.003 per 1K tokens

Bedrock Architecture & Runtime API

Key Bedrock Features

üöÄ Serverless

  • No Infrastructure: Zero servers to manage
  • Auto-scaling: Handles any load
  • High Availability: 99.9% SLA

ü§ñ Multi-Model

  • 5+ Providers: Anthropic, Meta, Cohere, Mistral, Amazon
  • Single API: Consistent interface
  • Easy Switching: Change models instantly

üîí Secure

  • Private: Data never leaves AWS
  • Encrypted: At rest and in transit
  • Compliant: HIPAA, SOC 2, GDPR

üîí Enterprise AI Security

VPC Endpoints, IAM & Data Protection

Bedrock Security Architecture

Security Best Practices

üîê IAM Least Privilege

  • Specific Models: Grant access to only needed models
  • Actions: InvokeModel only (not ListModels)
  • Conditions: Restrict by IP, VPC, time

üåê VPC Endpoints

  • Private Access: No internet gateway needed
  • PrivateLink: Secure AWS backbone
  • Cost: $0.01/hour + $0.01/GB

üîí Data Protection

  • Encryption: TLS 1.2+ in transit, AES-256 at rest
  • No Training: Your data never trains models
  • Compliance: HIPAA, SOC 2, GDPR ready

Code Examples

1. boto3 - Direct API Call

import boto3 import json bedrock = boto3.client('bedrock-runtime', region_name='us-east-1') prompt = "Explain the difference between savings and checking accounts" body = json.dumps({ "anthropic_version": "bedrock-2023-05-31", "max_tokens": 1024, "messages": [{"role": "user", "content": prompt}] }) response = bedrock.invoke_model( modelId='anthropic.claude-sonnet-4-5-20250929-v1:0', body=body ) response_body = json.loads(response['body'].read()) print(response_body['content'][0]['text'])

2. LangChain - Prompt Template & Chain

from langchain_aws import BedrockLLM from langchain.prompts import PromptTemplate from langchain.chains import LLMChain llm = BedrockLLM( model_id="anthropic.claude-sonnet-4-5-20250929-v1:0", region_name="us-east-1" ) template = """You are a helpful banking assistant for SecureBank. Customer Question: {question} Provide a clear, professional response:""" prompt = PromptTemplate(template=template, input_variables=["question"]) chain = LLMChain(llm=llm, prompt=prompt) response = chain.run(question="What is my account balance?") print(response)

3. LlamaIndex - Document Q&A

from llama_index.llms.bedrock import Bedrock from llama_index.core import VectorStoreIndex, SimpleDirectoryReader llm = Bedrock(model="anthropic.claude-sonnet-4-5-20250929-v1:0") # Load banking documents documents = SimpleDirectoryReader('banking_docs').load_data() # Create index index = VectorStoreIndex.from_documents(documents, llm=llm) # Query query_engine = index.as_query_engine() response = query_engine.query("What are the loan eligibility criteria?") print(response)
