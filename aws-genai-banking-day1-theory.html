Security Groups vs NACLs

üõ°Ô∏è Security Groups (Stateful)

  • Instance Level: Applied to ENIs
  • Allow Rules Only: No explicit deny
  • Stateful: Return traffic automatically allowed
  • All Rules Evaluated: Most permissive wins
# Banking App Security Group Port 443 (HTTPS) - Allow from ALB SG Port 8080 (App) - Allow from ALB SG only Port 3306 (MySQL) - Deny (handled by DB SG)

üöß NACLs (Stateless)

  • Subnet Level: Applied to all instances
  • Allow & Deny Rules: Explicit deny possible
  • Stateless: Must configure both directions
  • Rule Order: Lowest number processed first
# Data Tier NACL 100 - Allow HTTPS from App Tier (10.0.2.0/24) 200 - Allow MySQL from App Tier (10.0.2.0/24) 300 - Deny ALL other traffic
