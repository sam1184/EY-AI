Example Conversation: Apply for Loan

Multi-turn Conversation

  • User: "I want to apply for a loan"
  • Bot: "What type of loan? Personal, Home, or Auto?"
  • User: "Home loan"
  • Bot: "How much do you need?"
  • User: "$200,000"
  • Bot: "What's the purpose?"
  • User: "Buying a house"
  • Bot: "Application submitted! Reference: #12345"

Lex Components Used

  • Intent: ApplyLoan
  • Slots:
    • LoanType (Personal/Home/Auto)
    • Amount (Number)
    • Purpose (Text)
  • Fulfillment: Lambda creates loan application in DynamoDB
  • Response: Confirmation with reference number
